Timebox

    s or click/tap on names = shuffle names // t or + / - or click/tap on time upper right before "min" = change timer // n or click/tap headline "TIMEBOX" = edit names

    space or click/tap timer = restart timer // d or doubleclick anywhere = from dusk till dawn

    URL-parameters: [names=comma,separated,list] / [time=0..n] // Click/tap here: copy current settings as url-parameters to clipboard
