# TIMEBOX

Small app that can be used in daily stand-ups. 

## URL-Parameters

names       - comma separated list of names
time        - time in minutes

## Hotkeys

| key      | function      |
| -------- | ------------- |
| enter, r | restart timer |
| s        | shuffle names |
| f        | toggle fade   |
| n        | edit names    |
| t        | edit time     |
